Different Views of a Linear System of Equations

Row Picture

Take the linear system of equations (1.1) as an example. Each equation can be seen as a plane in 3D space, so, this system of equations represents three different planes, and the solution of the set will be the intersection point of these three planes. Here, the solution is x=−12,y=0,z=12x = - \frac{1}{2}, y = 0, z=\frac{1}{2}x=−21​,y=0,z=21​. I used Mathematica to visualize that (see Fig.1.1 and 1.2)

{x+y+z=0x+2y+3z=1x−4y+5z=2(1.1)\begin{cases} x + y + z=0 \\ x + 2y + 3z=1 \\ x - 4y + 5z=2 \end{cases} \tag{1.1} ⎩⎪⎨⎪⎧​x+y+z=0x+2y+3z=1x−4y+5z=2​(1.1)

In this simple case, we can easily find the solution with computer or even by drawing on the paper. However, if the number of equations and variables increase , the visualization will be hard or impossible for now. In 1.2, there are eight variables in eight equations. Each equation represents a 7D space 😕, which cannot be visualized.

{2x1+3x2+2x3−3x4−2x5+12x6+5x7−x8=−1−1x1−4x2+34x3+8x4−27x5+3x6+2x7+3x8=5−4x1+7x2+3x4−7x5+12x6+5x7+4x8=103x1+12x2+x3−14x4−5x5−3x6+8x7−11x8=−47x1−8x2+16x3+9x4−x5+9x6−5x7+6x8=−169x1+3x2−5x3+5x4+12x5+20x7+7x8=8−x1+6x2−3x3+21x4+2x5−3x6+13x7−x8=6174x1+78x2+4x3+3x4−3x5+7x6+5x7+3x8=−9(1.2)\begin{cases} 2x_1 + 3x_2 + 2x_3 - 3x_4 - 2x_5 + \frac{1}{2}x_6 + 5x_7 - x_8 = -1 \\[1ex] -1x_1 - 4x_2 + \frac{3}{4}x_3 + 8x_4 -\frac{2}{7}x_5 + 3x_6 + 2x_7 + 3x_8 = 5\\[1ex] -4x_1 + 7x_2 + 3x_4 - 7x_5 + \frac{1}{2}x_6 + 5x_7 + 4x_8 = 10\\[1ex] 3x_1 + 12x_2 + x_3 - 14x_4 - 5x_5 - 3x_6 + 8x_7 - 11x_8 = -4\\[1ex] 7x_1 - 8x_2 + 16x_3 + 9x_4 - x_5 + 9x_6 - 5x_7 + 6x_8 = -16\\[1ex] 9x_1 + 3x_2 - 5x_3 + 5x_4 + 12x_5 + 20x_7 + 7x_8 = 8\\[1ex] -x_1 + 6x_2 - 3x_3 + 21x_4 + 2x_5 - 3x_6 + 13x_7 - x_8 = \frac{6}{17}\\[1ex] 4x_1 + \frac{7}{8}x_2 + 4x_3 + 3x_4 - 3x_5 + 7x_6 + 5x_7 + 3x_8 = -9 \end{cases} \tag{1.2} ⎩⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎨⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎪⎧​2x1​+3x2​+2x3​−3x4​−2x5​+21​x6​+5x7​−x8​=−1−1x1​−4x2​+43​x3​+8x4​−72​x5​+3x6​+2x7​+3x8​=5−4x1​+7x2​+3x4​−7x5​+21​x6​+5x7​+4x8​=103x1​+12x2​+x3​−14x4​−5x5​−3x6​+8x7​−11x8​=−47x1​−8x2​+16x3​+9x4​−x5​+9x6​−5x7​+6x8​=−169x1​+3x2​−5x3​+5x4​+12x5​+20x7​+7x8​=8−x1​+6x2​−3x3​+21x4​+2x5​−3x6​+13x7​−x8​=176​4x1​+87​x2​+4x3​+3x4​−3x5​+7x6​+5x7​+3x8​=−9​(1.2)

Column Picture

Again, let's consider the system of equations (1.1). This time, each column of these equations are viewed as a column vector. That system of equations can be seen as a combination of column vectors.

{x+y+z=0x+2y+3z=1x−4y+5z=2⟶x[111]+y[12−4]+z[135]=[012]\begin{cases} x + \hspace{0.5em}y + \hspace{0.5em}z=0 \\ x + 2y + 3z=1 \\ x - 4y + 5z=2 \end{cases} \hspace{1em}\longrightarrow\hspace{1em} x \begin{bmatrix} 1\\ 1\\ 1 \end{bmatrix} +y \begin{bmatrix} 1\\ 2\\ -4 \end{bmatrix} +z \begin{bmatrix} 1\\ 3\\ 5 \end{bmatrix} = \begin{bmatrix} 0\\ 1\\ 2 \end{bmatrix} ⎩⎪⎨⎪⎧​x+y+z=0x+2y+3z=1x−4y+5z=2​⟶x⎣⎡​111​⎦⎤​+y⎣⎡​12−4​⎦⎤​+z⎣⎡​135​⎦⎤​=⎣⎡​012​⎦⎤​

The equations are transferred to linear combinations of vectors. Thus, solving the system of equations is to find suitable xxx, yyy and zzz so that the combination of the vectors on the left hand of equation equals to the vector on the right hand.

Compared with the row picture, it's easier to be understood in high dimension cases. For the system of equations (1.2), it can be viewed as a combination of eight 8D vectors (Those in the right hand), creating another vector in 8D space (The one in the right hand). No matter how high the dimension is, a vector can always be represented by an arrow. You don't need to imagine a 7D space.
